Эта библиотека может пригодиться тебе в случае проверки корректности ввода данных в поле input. Например поле: имя, email, номер телефона...библиотека будет развиваться.

Для того чтобы воспользоваться библиотекой её необходимо подключить.
ссылка для подлкючения
исходный код на GitHub

Например функция validationName() принимает в качестве параметра поле input userName и возвращает true || false

validationName(userName) - проверяет поле с именем пользователя

validationEmail(userEmail) - проверяет поле с почтой пользователя

validationNumberPhone(userPhone) - проверяет поле с номером телефона пользователя
